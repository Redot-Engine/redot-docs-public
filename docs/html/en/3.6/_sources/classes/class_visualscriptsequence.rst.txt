Description
-----------

Steps through a series of one or more output Sequence ports. The ``current`` data port outputs the currently executing item.

\ **Input Ports:**\ 

- Sequence: ``in order``\ 

\ **Output Ports:**\ 

- Sequence: ``1``\ 

- Sequence: ``2 - n`` (optional)

- Data (int): ``current``
