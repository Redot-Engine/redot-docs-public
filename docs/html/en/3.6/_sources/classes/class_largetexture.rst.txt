LargeTexture
============

**Inherits:** :ref:`Texture<class_Texture>` **<** :ref:`Resource<class_Resource>` **<** :ref:`Reference<class_Reference>` **<** :ref:`Object<class_Object>`

.. rst-class:: classref-introduction-group

Description
-----------

*Deprecated (will be removed in Redot 4.0).* A :ref:`Texture<class_Texture>` capable of storing many smaller textures with offsets.

You can dynamically add pieces (:ref:`Texture<class_Texture>`\ s) to this **LargeTexture** using different offsets.
